# CONTEXT.md — Clausius-Clapeyron 검증 실험

## 용어집

### 실험 장치
- **비커**: 1000mL 대형 비커, 물 약 0.7~0.85L 충전
- **메스실린더**: 10mL 정밀 유리, 외경 반지름 17mm, 높이 152mm, 내경 ≈ 1.01cm (추정). 거꾸로 뒤집어 비커 물에 담금
- **공기방울**: 메스실린더 상단(닫힌 끝)에 갇힌 기체 혼합물(공기+수증기)
- **"2.0mL 노출"** / **"2.2mL 노출"**: 비커 수면 위로 드러난 메스실린더 부피 공간. **각 실험 전체의 고정 초기 조건**이며 포인트마다 변하지 않음. 수압 보정용 h 계산에 사용
- **V_raw**: 메스실린더 눈금 직독값 = 공기방울 부피. 역방향(10mL=하단)
- **V_corr**: meniscus 보정 후 기체 부피 = V_raw - 0.2mL

### 물리량
- **P_gas**: 갇힌 기체의 실제 압력 = P_ATM - ρgh (수압 보정 포함). 가열/냉각 각각 고정
- **h**: V_exp / A (메스실린더 단면적). 가열: 2.0/0.8012 ≈ 2.50cm, 냉각: 2.2/0.8012 ≈ 2.75cm
- **n_air**: 갇힌 공기의 몰수. **독립 scanning 파라미터** (초기조건 종속 해제)
- **P_water**: 순수 수증기압 = P_gas - P_air
- **δT**: 열 불균형 보정 상수. T_gas = T_water - δT. **독립 scanning 파라미터**

### 핵심 방정식
- Dalton: P_air + P_water = P_gas = P_ATM - ρgh
- Ideal gas: P_air = n_air · R · T_gas / V_corr
- C-C: ln(P_water) = -(ΔH_vap / R) × (1/T_gas) + C
- T_gas = T_water - δT

### 보정 파이프라인 설계 원칙
1. **Raw 데이터 우선**: 측정된 온도-부피 원시데이터 + δT 물리 모델 직접 적용
2. **독립 변수 scanning**: δT와 n_air 모두 초기조건 종속 없이 독립적으로 최적화
3. **P_water > 0 제약**: 모든 데이터 포인트에서 물리적 모순 배제
4. **가열·냉각 통합**: n_air는 가열/냉각 각각 독립 scanning. 냉각 저온 포인트 품질 불충분 시 가열 데이터만 사용
5. **이중 종료 조건**: ΔH_vap ≈ 40.7 kJ/mol + 삼중점 P ≈ 0.006 atm

### 수정 이력
- V_exp: 각 포인트마다 변하는 값이 아니라 실험 전체 고정값 (가열=2.0mL, 냉각=2.2mL)
- n_air: 초기조건 Dalton 계산에서 δT 피드백 루프 제거. 독립 scanning 파라미터로 승격
- 냉각 포인트 3,4 (둘 다 13°C, 2.8mL): 중복이므로 1개만 사용
