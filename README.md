# Clausius-Clapeyron 검증 실험

온도 변화에 따른 기체 부피 데이터를 바탕으로 물의 증발 엔탈피(ΔH_vap)를 구하여 클라우지우스-클라페이롱 식을 검증하고, 데이터가 없는 저온 영역을 외삽(Extrapolation)하여 물의 삼중점 압력을 추정하는 프로젝트.

## 핵심 문제의식

고온 구간(21~62°C) 데이터만으로 0°C 영역을 외삽할 때, C-C 회귀의 **기울기(ΔH_vap)**와 **절편(삼중점 P)**이 트레이드오프 관계에 있다. ΔH_vap를 문헌값에 맞추면 삼중점이 과대평가되고, 삼중점을 맞추면 ΔH_vap가 과대평가된다.

이 딜레마는 실험 데이터의 본질적 한계이며, **비교 분석(Comparative Analysis)** 을 통해 해결의 실마리를 찾는다.

## 분석 케이스

| Case | 데이터 | 포인트 | 설명 |
|------|--------|--------|------|
| A | 가열만 | 12 | 21~62°C, 고온 구간만 |
| B | 가열 + 냉각 20°C | 13 | Case A에 냉각 안정 포인트 1개 추가 |
| C | 가열 + 냉각 전체 (P_water>0) | 가변 | P_water>0 물리적 제약 만족 포인트만 |
| D | 가열+냉각 외삽 앵커 | 13+가중치 | Case A에 냉각 0°C 외삽 앵커 가중 포함 |

## 독립 변수 (Scanning)

- **δT**: 열 불균형 보정 (T_gas = T_water - δT, 상수)
- **n_air_heat**: 가열 실험 공기 몰수 (독립)
- **n_air_cool**: 냉각 실험 공기 몰수 (독립, 새로 따서 넣었으므로 가열과 다름)

## 종료 조건

- ΔH_vap ≈ 40.7 kJ/mol (문헌값)
- 삼중점 P ≈ 0.00603 atm (문헌값)
- P_water > 0 (모든 포인트, 물리적 제약)
