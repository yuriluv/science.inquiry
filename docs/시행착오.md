# 시행착오 기록

## v1 (초기, 폐기)
- n_air를 "2.0mL 노출"을 공기 부피로 해석하여 계산 → 오류 발견
- 수압 보정 ρgh를 각 데이터 포인트마다 다른 V_exp로 계산 → V_exp는 실험 고정값이었음
- 냉각 포인트 3,4가 동일 조건(13°C, 2.8mL)인데 중복으로 포함 → 제거 필요

## v2 (δT 단일 scanning, 폐기)
- δT만 scanning, n_air는 초기조건 Dalton 법칙으로 종속 계산
- δT↑ → n_air↑ 피드백 루프 발견: T_gas↓ → P_water_lit↓ → P_air↑ → n_air↑
- 이 피드백으로 냉각 저온 포인트 P_water < 0 모순 심화
- δT ≈ 4.0에서 회귀 불연속 점프 발생 (냉각 포인트 탈락으로 인한)

## v3 (δT + n_air 독립 scanning, 현재)
- δT, n_air 모두 독립 scanning 파라미터로 승격
- P_water > 0 물리적 제약 추가
- V_exp 고정값 수정 (가열=2.0mL, 냉각=2.2mL)
- 냉각 포인트 3,4 중복 제거 (13°C, 2.8mL 1개만 사용)

### v3 결과 (가열 12포인트만)
- ΔH_vap와 삼중점 P 사이 트레이드오프 존재
- δT=1.6, n=1.06e-4: ΔH=45.0(오차10.5%), 삼중점=0.00612(오차1.5%)
- δT=3.2, n=1.06e-4: ΔH=40.9(오차0.4%), 삼중점=0.0091(오차51%)
- δT=3.6, n=1.07e-4: ΔH=46.4(오차14%), 삼중점=0.00603(오차0%)
- 두 목표를 5% 오차 내로 동시 만족 불가 (가열만으로는 외삽 한계)
