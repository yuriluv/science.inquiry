# Clausius-Clapeyron 검증 실험 — 결과 보고서

## 1. 실험 개요

밀폐 메스실린더 내 물과 공기의 혼합 기체 온도-부피 관계를 측정하여, Clausius-Clapeyron(C-C) 방정식을 검증하고 물의 증발 엔탈피(ΔH_vap)와 삼중점 압력(P_triple)을 유도하는 실험.

### C-C 방정식

```
ln(P_water) = -(ΔH_vap / R) × (1/T) + C
```

- 기울기 = -ΔH_vap / R → ΔH_vap = -기울기 × R
- 절편 C → 삼중점: P_triple = exp(기울기 / T_triple + C)

### 실험 원리

밀폐 메스실린더에서 기체 전체 압력(P_gas)은 일정:

```
P_gas = P_ATM - ρgh (수압 보정)
P_gas = P_air + P_water (Dalton 분압 법칙)
P_air = n_air × R × T_gas / V (이상기체)
P_water = P_gas - P_air (잔여 분압 = 수증기압)
```

## 3. 보정 파라미터

| 파라미터 | 역할 | 처리 방식 |
|-----------|------|-----------|
| δT | 열 불균형 (T_gas = T_water - δT) | 독립 scanning |
| n_air_가열 | 가열 실험 공기 몰수 | 독립 scanning |
| n_air_냉각 | 냉각 실험 공기 몰수 | 독립 scanning (가열과 별도) |
| V_exp | 수압 보정용 기포 높이 | 고정 (가열=2.0, 냉각=2.2 mL) |

### n_air 독립 이유

초기에는 Dalton 법칙으로 n_air를 초기조건에서 계산했으나, δT↑ → T_gas↓ → P_water_lit↓ → P_air↑ → n_air↑ 피드백 루프가 발견됨. 특히 냉각 구간에서 δT↑ → n_air↑ → P_air↑ → P_water < 0 모순 발생. 따라서 n_air는 독립 scanning 파라미터로 승격.

## 4. 분석 방법

4가지 Case를 비교 분석:

### Case A: 가열 데이터만 (12포인트)
- 고온 구간(21~62°C) 데이터만으로 C-C 회귀
- δT, n_air_heat scanning

### Case B: 가열 + 냉각 20°C (13포인트)
- Case A에 냉각 안정 포인트(20°C) 1개 추가
- δT, n_air_heat, n_air_cool scanning

### Case C: 가열 + 냉각 전체 (P_water > 0 만족 포인트)
- 가열 12 + 냉각 3 = 최대 15포인트
- P_water > 0 물리적 제약 만족 포인트만 포함

### Case D: 가열 + 냉각 외삽 앵커 보정
- 냉각 데이터로 독자적 C-C 회귀 → 0°C 삼중점 외삽
- 외삽값을 앵커 포인트로 가열 회귀에 가중 포함 (WLS)
- 앵커 가중치 w = 0.5 ~ 5.0 민감도 분석

## 6. 논의

### 6.1 열 불균형 보정 δT의 물리적 의미

δT는 기체 온도(T_gas)가 측정 수온(T_water)보다 낮은 정도를 나타낸다. 최적 δT ≈ 0.7~1.6°C로, 실험 조건에서 기체가 물보다 약간 차갑다는 것과 일치한다. 이는 기체가 수온과 평형에 도달하기 전의 지연, 메스실린더 벽을 통한 열손실 등으로 설명된다.

### 6.2 n_air의 독립성

가열과 냉각 실험에서 공기 방울이 서로 다른 양을 포함할 수 있다는 전제가 물리적으로 합리적:
- 가열과 냉각은 별도 실행 → 공기 방울 양이 다를 수 있음
- 온도 변화에 따른 공기 방울 체적 변화는 이상기체 법칙으로 이미 모델링됨
- 그러나 n_air가 δT와 양의 상관관계를 가지면, 냉각 저온(δT 효과 큼)에서 P_water < 0 위반 위험

### 6.3 냉각 데이터의 한계

냉각 데이터 3포인트(20, 14, 13°C)는 C-C 회귀에 다음 문제를 야기:
1. **P_water 과대**: n_air가 약간만 커져도 저온에서 P_air가 P_gas를 초과
2. **온도 범위 협소**: 7°C 폭(13~20°C)으로 회귀에 정보 부족
3. **동적 평형 의문**: 냉각 과정에서 기체-액체 평형에 도달했는지 불확실

### 6.4 앵커 보정법(Case D)의 의의

냉각 데이터를 직접 회귀에 포함하는 것은 실패했으나, 냉각 데이터로 얻은 0°C 외삽값을 앵커로 사용하면:
- 가열 데이터의 R²를 0.978 → 0.994로 개선
- 삼중점 오차를 1.5% → 0.1%로 극적 개선
- C-C 그래프의 저온 외삽 방향을 물리적으로 올바르게 유도

이는 **데이터의 직접 결합보다 정보의 간접 활용이 유효할 수 있음**을 보여준다.

### 6.5 ΔH_vap 8.5~10.5% 오차의 원인

ΔH_vap가 문헌값(40.7 kJ/mol)보다 일관되게 과대(44~45 kJ/mol)인 이유:
1. **δT 보정 불완전**: T_gas = T_water - δT 모델이 단순 상수이나, 실제로는 온도 의존적일 수 있음
2. **메니스커스 보정**: 0.2 mL는 근사값, 실제 곡면 효과는 온도에 따라 변할 수 있음
3. **수압 보정**: V_exp 고정값(2.0 mL)에 측정 오차가 포함될 수 있음
4. **기체 비이상성**: 1 atm에서 이상기체 근사가 약간 어긋날 수 있음

## 7. 결론

1. **ΔH_vap ≤5% AND 삼중점 ≤5% 동시 달성은 실험 데이터의 본질적 한계**로, 어떤 보정 방법으로도 불가능하다.

2. **가열 데이터만(Case A)** 으로 ΔH_vap = 44.98 kJ/mol (오차 10.5%), 삼중점 0.00612 atm (오차 1.5%)를 얻는다. R² = 0.978로 선형성은 우수하나 ΔH_vap 과대평가가 피할 수 없다.

3. **냉각 외삽 앵커 보정(Case D)** 이 가장 균형 잡힌 결과를 제공한다: ΔH_vap = 44.17 kJ/mol (오차 8.5%), 삼중점 0.00602 atm (오차 0.2%), R² = 0.994.

4. 이 **트레이드오프 현상 자체**가 C-C 방정식 검증 실험에서 고온 구간만으로 저온 외삽 시 발생하는 체계적 편향의 실례이며, 실험적 한계를 명확히 보여준다.
